Semana del 9 de febrero, 2026

AMENAZA MEDIA

Orchid Security Introduces Continuous Identity Observability for Enterprise Applications

Resumen:

Orchid Security presentó una plataforma de observabilidad continua de identidades que detecta el uso de identidades dentro de aplicaciones (credenciales embebidas, cuentas de servicio y autenticaciones personalizadas), analiza el comportamiento para priorizar riesgos reales (credenciales hardcodeadas, cuentas huérfanas, rutas privilegiadas sin gestionar), orquesta la remediación vía flujos IAM/PAM existentes y mantiene evidencia continua lista para auditoría.

Puntos clave:

  • Descubrimiento automático de identidades y credenciales embebidas dentro de aplicaciones.
  • Análisis de comportamiento para priorizar riesgos reales y rutas privilegiadas no gestionadas.
  • Integración con procesos IAM/PAM para orquestar remediación y conservar evidencia de auditoría.

Detalles adicionales:

Recomendado evaluar la integración con tus pipelines de CI/CD y plantillas/base images, revisar cuentas de servicio y credenciales hardcodeadas, y aprovechar las evidencias continuas para auditorías y cumplimiento. Aunque no es una explotación activa, mejora la visibilidad y reduce riesgo de abuso de identidades.

AMENAZA ALTA

Mutagen Astronomy: From Discovery to CISA Recognition—A Seven-Year Journey

Resumen:

CISA añadió la vulnerabilidad CVE-2018-14634 ("Mutagen Astronomy"), un escalamiento de privilegios local en el kernel de Linux descubierto por Qualys en 2018, a su catálogo de Vulnerabilidades Conocidas Explotadas (KEV) tras confirmarse explotación en el mundo real. Las organizaciones deben priorizar parches en distribuciones afectadas, actualizar imágenes base y verificar la remediación.

Puntos clave:

  • CVE-2018-14634 es un LPE (Local Privilege Escalation) en kernel de Linux con explotación confirmada en campo.
  • Incorporada al catálogo KEV de CISA, lo que implica prioridad para mitigación en entornos críticos.
  • Acciones recomendadas: parchear distribuciones, actualizar imágenes base/plantillas y validar la corrección.

Detalles adicionales:

Además de aplicar parches, audita imágenes de contenedor y máquinas virtuales para detectar kernels vulnerables, implemente controles compensatorios (hardening, restricciones de acceso) y valide mediante escaneos y pruebas de penetración internas para asegurar que no queden sistemas sin mitigar.

AMENAZA ALTA

Docker Fixes Critical Ask Gordon AI Flaw Allowing Code Execution via Image Metadata

Resumen:

Docker parcheó una falla crítica en Ask Gordon (el asistente AI de Docker Desktop/CLI) descubierta por Noma Labs: metadatos LABEL maliciosos en imágenes Docker podían inyectar instrucciones que Ask Gordon reenviaba al MCP Gateway, permitiendo la ejecución de herramientas MCP sin validación y posibilitando ejecución remota de código o exfiltración. El arreglo está en Docker Desktop/CLI versión 4.50.0.

Puntos clave:

  • Vector: metadatos LABEL en imágenes Docker que manipulan el contexto entregado al asistente AI.
  • Impacto: posibilidad de ejecución remota de comandos y exfiltración vía MCP Gateway si no se valida el contexto.
  • Mitigación: actualizar a Docker Desktop/CLI 4.50.0 y aplicar validación de entrada/contexto AI siguiendo principios de zero-trust.

Detalles adicionales:

Se recomienda aplicar el parche inmediatamente, restringir privilegios de ejecución desde asistentes AI, validar cualquier instrucción que provenga de contexto automatizado, y monitorear registros de MCP Gateway y comportamiento anómalo en entornos donde se utilice Ask Gordon.

AMENAZA ALTA

2025 Q4 DDoS threat report: A record-setting 31.4 Tbps attack caps a year of massive DDoS assaults

Resumen:

Cloudflare reporta un aumento masivo en 2025 con 47.1 millones de ataques DDoS (121% más), destacando el botnet Aisuru‑Kimwolf que lanzó ataques HTTP hiper‑volumétricos de hasta 205 Mrps y picos de decenas de Tbps; proveedores telco, gaming y servicios en la nube fueron objetivos principales y las defensas de Cloudflare mitigaron estos ataques.

Puntos clave:

  • Incremento del 121% en cantidad de ataques DDoS reportados en 2025.
  • Botnet Aisuru‑Kimwolf ejecutó ataques HTTP a gran escala (hasta cientos de Mrps y Tbps).
  • Sectores afectados: telecomunicaciones, gaming y servicios cloud; mitigaciones a gran escala fueron necesarias.

Detalles adicionales:

Organizaciones deben revisar estrategias de mitigación DDoS (CDN, scrubbing, rate limiting, arquitectura redundante), probar procedimientos de respuesta a incidentes y coordinarse con proveedores de red y seguridad para asegurar escalabilidad ante ataques volumétricos.

AMENAZA ALTA

DEAD#VAX Malware Campaign Deploys AsyncRAT via IPFS-Hosted VHD Phishing Files

Resumen:

Investigadores describen la campaña DEAD#VAX que entrega AsyncRAT mediante archivos VHD alojados en IPFS, disfrazados como PDFs. El ataque multinivel usa WSF, scripts batch y loaders en PowerShell fuertemente ofuscados para descifrar e inyectar AsyncRAT como shellcode en memoria dentro de procesos Microsoft confiables, además de persistencia vía tareas programadas y técnicas sin archivos para evadir detección y complicar el análisis forense.

Puntos clave:

  • Entrega: archivos VHD en IPFS usados en phishing, con apariencia de PDFs.
  • Técnicas: loaders ofuscados, inyección en memoria (fileless) y persistencia mediante tareas programadas.
  • Payload: AsyncRAT desplegado como shellcode en procesos confiables para evadir detección.

Detalles adicionales:

Mitigaciones recomendadas: bloquear/monitorizar acceso a dominios y gateways IPFS sospechosos, restringir ejecución de scripts (WSF/PowerShell), aplicar detección de comportamiento en memoria con EDR, auditar tareas programadas y educar a usuarios sobre phishing con archivos VHD/adjuntos disfrazados.
