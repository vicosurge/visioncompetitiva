Resumen Semanal de Ciberseguridad

Semana del 24 de octubre, 2025

Mantente al día con las últimas amenazas, vulnerabilidades y desarrollos en ciberseguridad. Nuestro resumen semanal te brinda información clave de fuentes confiables para proteger tu organización.

AMENAZA ALTA

131 Chrome Extensions Caught Hijacking WhatsApp Web for Massive Spam Campaign

Resumen:

Investigadores identificaron 131 clones rebrandeados de una extensión de automatización para WhatsApp Web que comparten la misma base de código, patrones e infraestructura, con aproximadamente 20,905 usuarios activos. Estas extensiones, comercializadas mediante un modelo white‑label (DBX Tecnologia) y publicadas mayormente por "WL Extensão", automatizan envíos masivos para evadir límites y controles anti‑spam; la actividad se mantiene desde hace meses con actualizaciones recientes.

Puntos clave:

  • 131 extensiones clonadas con la misma infraestructura maliciosa.
  • ~20,905 usuarios activos expuestos a campañas masivas de spam.
  • Diseñadas para evadir límites y controles anti‑spam de WhatsApp Web.

Detalles adicionales:

La campaña está en operación activa; se recomienda auditar extensiones instaladas, eliminar las sospechosas, bloquear dominios/infrastructura asociados y denunciar las entradas maliciosas en Chrome Web Store. Implementar detección de actividad inusual en cuentas y revisar automatizaciones que interactúan con WhatsApp Web.

AMENAZA ALTA

Vidar Stealer 2.0 adds multi-threaded data theft, better evasion

Resumen:

Vidar Stealer 2.0, reescrito en C, incorpora recolección de datos multi‑hilo, técnicas avanzadas anti‑análisis y polimorfismo, además de evasión que sortea cifrados ligados a apps de Chrome mediante inyección en memoria. Apunta a navegadores, monederos, cuentas en la nube y redes sociales, y exfiltra información a bots de Telegram y URLs de Steam; los investigadores esperan un incremento en su actividad.

Puntos clave:

  • Reescrito en C con soporte multi‑hilo para exfiltración más rápida.
  • Técnicas anti‑análisis y polimorfismo que complican la detección.
  • Exfiltra credenciales y datos a bots de Telegram y enlaces de Steam.

Detalles adicionales:

La evolución del malware dificulta su identificación por firmas tradicionales; se recomienda reforzar EDR/AV con detección de comportamiento, monitorear conexiones a servicios de exfiltración (Telegram/URLs sospechosas), forzar MFA y rotar credenciales comprometidas.

AMENAZA ALTA

Cursor, Windsurf IDEs riddled with 94+ n-day Chromium vulnerabilities

Resumen:

Las versiones recientes de los IDEs Cursor y Windsurf empaquetan componentes obsoletos de Electron/Chromium/V8, exponiendo a más de 94 vulnerabilidades conocidas y parchadas. Investigadores de Ox Security demostraron un PoC para CVE‑2025‑7656 (DoS) y advierten riesgo de ejecución remota mediante extensiones maliciosas, previews de README o phishing. Cursor calificó el reporte como "fuera de alcance" y Windsurf no respondió; alrededor de 1.8 millones de desarrolladores podrían verse afectados.

Puntos clave:

  • Más de 94 vulnerabilidades n‑day en componentes Chromium/V8 empaquetados.
  • PoC publicado para CVE‑2025‑7656 (DoS); posibilidad de RCE mediante vectores auxiliares.
  • Impacto potencial amplio entre desarrolladores y proyectos que usan estos IDEs.

Detalles adicionales:

Se recomienda no usar versiones vulnerables en entornos sensibles, deshabilitar extensiones no confiables, aislar el uso del IDE y exigir actualizaciones del proveedor. Si no hay parche inmediato, evaluar alternativas o mitigar mediante políticas de seguridad y sandboxes.

AMENAZA ALTA

A Strategic Response to the F5 BIG-IP Nation-State Breach

Resumen:

Una brecha atribuida a un actor de estado‑nación exfiltró código fuente de BIG‑IP y detalles de vulnerabilidades sin parche, lo que motivó una advertencia de CISA sobre una amenaza inminente. Qualys recomienda acciones urgentes: inventariar activos F5, parchear sistemas expuestos, endurecer planos de gestión, auditar configuraciones y buscar indicadores de compromiso; también ofrece herramientas de descubrimiento y VMDR para asistir en la respuesta.

Puntos clave:

  • Exfiltración de código fuente y detalles de vulnerabilidades de BIG‑IP.
  • CISA advierte riesgo inminente para sistemas sin parchear.
  • Medidas urgentes: inventario, parcheo, endurecimiento y búsqueda de compromisos.

Detalles adicionales:

Organizaciones con dispositivos F5 deben priorizar la detección y mitigación: aplicar parches, rotar credenciales, restringir acceso administrativo, revisar logs y desplegar escaneos de vulnerabilidades continuos para identificar exposición y compromisos.

AMENAZA MEDIA

Microsoft fixes bug preventing users from opening classic Outlook

Resumen:

Microsoft corrigió un error que impedía a usuarios de Microsoft 365 abrir el cliente clásico de Outlook en Windows. El artículo de soporte fue actualizado para indicar que el problema conocido está resuelto y el equipo de Outlook está monitoreando la implementación. Usuarios aún afectados deben usar Outlook Web Access o el nuevo Outlook para Windows; Microsoft también publica pasos de solución (modo seguro, nuevo perfil, reparación de archivos de datos, /resetnavpane).

Puntos clave:

  • Bug que impedía iniciar Outlook clásico fue corregido por Microsoft.
  • Solución temporal: usar OWA o el nuevo Outlook para Windows.
  • Pasos de resolución disponibles: modo seguro, nuevo perfil, reparación y /resetnavpane.

Detalles adicionales:

El equipo de Outlook monitorea la corrección; si persisten problemas, seguir las guías de soporte y contactar al soporte técnico de Microsoft. Revisar despliegues y actualizaciones en entornos gestionados para asegurar que la corrección se haya aplicado a todos los usuarios.
